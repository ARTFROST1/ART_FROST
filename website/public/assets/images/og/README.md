# OG Images (Open Graph)

Эта папка содержит OG-изображения для красивых превью в соцсетях и мессенджерах.

## Требования

- **Размер:** 1200 × 630 px (обязательно!)
- **Формат:** PNG или JPG
- **Размер файла:** < 300 KB
- **Safe zone:** 1080 × 560 px (центральная область не обрезается)

## Требуемые файлы (MVP)

1. **default.png** — Главная страница (/)
   - Заголовок: "Art Frost"
   - Подзаголовок: "IT-специалист & Vibe Coder"

2. **about.png** — Страница "Обо мне" (/about)
   - Заголовок: "Обо мне"
   - Подзаголовок: "Узнайте больше о моём опыте"

3. **projects.png** — Список проектов (/projects)
   - Заголовок: "Портфолио"
   - Подзаголовок: "Мои последние проекты"

4. **links.png** — Страница Links (/links)
   - Заголовок: "Все ссылки"
   - Подзаголовок: "Найдите меня в соцсетях"

### Элементы дизайна

1. **Фон:** Градиент `#050605` → `#10B981`
2. **Логотип:** Верхний левый угол (32×32 — 48×48 px)
3. **Заголовок:** 48–72px, bold, белый
4. **Подзаголовок:** 24–32px, regular, серый (#A1A1AA)
5. **URL:** Внизу справа, 18px, muted color
6. **Отступы:** Минимум 60px от краёв

## Как добавить

1. Создай OG-изображение в Figma/Canva/Photoshop
2. Экспортируй в PNG (1200×630 px)
3. Оптимизируй через `pngquant` и `oxipng`
4. Положи в эту папку (`website/public/assets/images/og/`)
5. Укажи путь в PageLayout:

```astro
<PageLayout
  title="О себе"
  description="..."
  image="/assets/images/og/about.png"  ← Путь к OG-изображению
/>
```

## Проверка превью

После добавления проверь через:

- **[opengraph.xyz](https://www.opengraph.xyz/)** — все платформы
- **Telegram** — вставь ссылку в чат
- **[Facebook Debugger](https://developers.facebook.com/tools/debug/)**
- **[Twitter Card Validator](https://cards-dev.twitter.com/validator)**
